## 🚀 Pour Démarrer (3 Étapes!)

### ⚡ Étape 1: Configurer Supabase
```
1. https://supabase.com → Sign up
2. Créer projet "alcootest"
3. Settings → API → Copier credentials
```

### ⚡ Étape 2: Créer les Tables
```
Dans Supabase SQL Editor:
- Collez le script SQL fourni
- Cliquez "Run"
- ✅ Fait!
```

### ⚡ Étape 3: Lancer l'App
```bash
npm install         # Dépendances
npm start          # Serveur :4200
# → Ouvrer http://localhost:4200
```

**Temps total: 5 minutes! ⚡**
